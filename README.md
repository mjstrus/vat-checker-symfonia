# Weryfikator VAT — Symfonia vs JPK_FA

Narzędzie do sprawdzania zgodności rejestru VAT z Symfonii FK z plikami JPK_FA z Saldeo.

## Jak używać

1. Pobierz rejestr VAT z Symfonii Finanse i Księgowość (XLSX)
2. Pobierz plik JPK_FA z zakładki "Dokumenty" w Saldeo
3. Wgraj oba pliki do aplikacji
4. Wybierz typ rejestru: **Zakup** (naliczony VAT) lub **Sprzedaż** (należny VAT)
5. Kliknij "Analizuj i porównaj"

## Co sprawdza

- ✅ Numery fakttur
- ✅ Daty dokumentów
- ✅ Kwoty (netto, VAT, brutto)
- ✅ NIP kontrahentów
- ✅ Status KSeF
- 🏛 Weryfikacja Białej Listy MF

Wyniki można wyeksportować do CSV.
